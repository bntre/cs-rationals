# cs-rationals

*C# development in various directions*

This project serves as a small experimental lab for exploring the intersections of math, music, and visuals.  
The library was originally created to explore music theory, specifically Just Intonation. Later, it was extended with MIDI support, WAV sound synthesis, and a lightweight graphics library for working with primitives, used for synchronized audio-visualization.

Showable results:

* [Rationals Explorer - Just intonation and temperament visualization tool](RationalsExplorer.md)
* [Xenharmonic hardcore - visualization of a transition between 12edo and 19edo tuning systems, combining beats and notes (2025)](XenharmonicHardcore.md)
* [TriTree - Search tree traversal auralization (2021)](samples/TriTree)
* [Tower of Hanoi solution visualization and auralization (2020)](samples/TowerOfHanoi)
