# Xenharmonic Hardcore

This demo shows how harmonic space transforms between the 12edo and 19edo tuning systems.  
A transition between 12edo and 19edo is demonstrated using simple pitch sequences in both systems. A subharmonic rhythm is employed in hopes of making the harmonic shifts more perceptible.
Everything is synchronously visualized using the [Grid module](RationalsExplorer.md#grid-structure).
